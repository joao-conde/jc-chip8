# Web CHIP-8 Emulator

Utilizes the [`jc-chip8`](../../jc-chip8/) crate compiled to [WebAssembly](https://webassembly.org/) to build a web application.

## Running

Serve the contents of this folder with any HTTP server and open `index.html`.

## Controls

Keys available: 

`1 2 3 4`

`Q W E R`

`A S D F`

`Z X C V`
