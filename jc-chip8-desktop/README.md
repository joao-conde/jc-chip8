# CHIP-8 Desktop Emulator

Uses the [`jc-chip8`](../jc-chip8/) crate and the [SDL crate](https://github.com/Rust-SDL2/rust-sdl2) to build a desktop emulator.

## Building

```
$ cargo build --release
```

## Running

```
$ cargo run --release
```

## Controls

Drag and drop a ROM file to play.

Keys available:

`1 2 3 4`

`Q W E R`

`A S D F`

`Z X C V`
