# CHIP-8 Web Emulator

Uses the [`jc-chip8`](../jc-chip8/) crate compiled to [WebAssembly](https://webassembly.org/) and the [Salvo crate](https://crates.io/crates/salvo/0.46.0) to serve the web emulator.

## Building

```
$ cargo build --release
```

## Running

```
$ cargo run --release
```

## Controls

Select a ROM to play.

Keys available:

`1 2 3 4`

`Q W E R`

`A S D F`

`Z X C V`
